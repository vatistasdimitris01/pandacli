# pandacli

适用于 [pandacli](https://github.com/vatistasdimitris01/pandacli) 的 CLI 工具。

## 安装

```bash
curl -fsSL https://raw.githubusercontent.com/vatistasdimitris01/pandacli/main/install.sh | bash
```

## 快速开始

```bash
pandacli --help
```

## 从源码构建

```bash
git clone https://github.com/vatistasdimitris01/pandacli.git
cd pandacli
bun install
bun run packages/opencode/src/index.ts --help
```
